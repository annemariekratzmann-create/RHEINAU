# Bewerbungswebsite – Annemarie Kratzmann

Persönliche Bewerbungswebsite für die Stelle **Mitarbeiterin (m/w/d) für den Bereich Digitalisierung** bei der **Stadt Rheinau**.

## Über diese Website

Eine einseitige, vollständig responsive Bewerbungswebsite, die Werdegang, Kompetenzen und Motivation für die ausgeschriebene Stelle präsentiert.

**Inhalte:**
- Kurzprofil & Motivation
- Beruflicher Werdegang & Ausbildung
- Kompetenzen & Fachkenntnisse
- Bachelorarbeit: KI-basierte Prozessoptimierung
- Warum Stadt Rheinau?
- Persönlichkeit & Haltung
- Kontakt

## Technik

- Reines HTML/CSS/JavaScript — keine Frameworks, keine Dependencies
- Alle Assets (Schriften, Bilder) inline eingebettet
- Vollständig offline nutzbar
- Responsive für Desktop & Mobile
